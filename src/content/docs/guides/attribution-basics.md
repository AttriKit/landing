## 什么是归因？

**归因（Attribution）** 是确定哪个营销触点（广告、渠道、活动）对用户转化行为负责的过程。

简单来说，归因回答了一个问题：**"用户是从哪里来的？"**

## 为什么归因很重要？

### 优化广告投放
- 知道哪些渠道带来高质量用户
- 停止浪费预算在低效渠道上
- 将预算分配给高 ROI 活动

### 提升转化率
- 了解用户转化路径
- 识别转化漏斗中的瓶颈
- 优化关键触点体验

### 数据驱动决策
- 基于真实数据做决策
- 减少主观判断偏差
- 持续迭代优化策略

## 归因模型

AttriKit 支持多种归因模型：

### 最后点击归因（Last Click）

将 100% 的转化价值归功于用户最后一次点击的广告。

```
用户路径：搜索广告 → 信息流广告 → 直接访问
                    ↑
                  归功于此
```

**适用场景**：直接转化、短决策周期

### 首次点击归因（First Click）

将 100% 的转化价值归功于用户第一次点击的广告。

```
用户路径：搜索广告 → 信息流广告 → 直接访问
      ↑
    归功于此
```

**适用场景**：品牌认知、新客户获取

### 线性归因（Linear）

将转化价值平均分配给路径上的所有触点。

```
用户路径：搜索广告(33%) → 信息流广告(33%) → 直接访问(33%)
```

**适用场景**：长转化周期、多触点协同

### 时间衰减归因（Time Decay）

越接近转化的触点获得越多的权重。

```
用户路径：搜索广告(20%) → 信息流广告(30%) → 直接访问(50%)
                                  ↑
                              权重最高
```

**适用场景**：决策周期短、再营销重要

### 位置归因（Position Based）

首次和最后一次触点各占 40%，中间触点占 20%。

```
用户路径：搜索广告(40%) → 信息流广告(20%) → 直接访问(40%)
      ↑                                        ↑
   引入兴趣                                最终转化
```

**适用场景**：品牌 + 效果兼顾

## 归因窗口

归因窗口是指从点击/曝光到发生转化的时间范围。

| 窗口类型 | 说明 | 推荐设置 |
|----------|------|----------|
| 点击窗口 | 点击广告后的转化时间 | 7-30 天 |
| 曝光窗口 | 看到广告后的转化时间 | 1-7 天 |
| 观看窗口 | 观看视频后的转化时间 | 1-3 天 |

## 转化类型

### 微转化
- 页面浏览
- 按钮点击
- 表单开始填写
- 视频播放

### 宏转化
- 表单提交
- 注册完成
- 加入购物车
- 购买成功

## 归因挑战

### 跨设备归因
用户可能在手机上看到广告，在电脑上完成转化。

**解决方案**：用户 ID 关联、登录识别

### 隐私保护
Cookie 限制、IDFA 政策收紧。

**解决方案**：第一方数据、概率归因

### 欺诈流量
虚假点击、刷量行为。

**解决方案**：异常检测、IP 黑名单

## 最佳实践

1. **选择适合的归因模型**
   - 电商推荐最后点击
   - B2B 推荐线性归因

2. **设置合理的归因窗口**
   - 高客单价商品延长窗口
   - 低客单价商品缩短窗口

3. **定义清晰的转化事件**
   - 区分微转化和宏转化
   - 设置转化价值

4. **定期验证数据准确性**
   - 对比多个数据源
   - 排查异常数据
