个人中心是每个用户管理自己账号信息和安全设置的专属页面。通过个人中心，用户可以更新个人信息、修改登录密码，确保账号安全和信息的准确性。

个人中心提供用户自我管理的核心功能，让用户无需管理员协助即可完成常见的账号维护操作。

## 操作指南

### 步骤一：进入个人中心

1. 登录 AttriKit Web 后台
2. 点击页面右上角的个人头像
3. 在下拉菜单中选择「个人中心」
4. 进入个人中心页面

### 步骤二：查看个人信息

个人中心页面展示您的当前账号信息：

- **用户名** - 您的登录账号
- **角色** - 您当前的权限角色（Admin/User）
- **创建时间** - 账号创建的日期
- **邮箱** - 您的邮箱地址（如果已设置）

### 步骤三：更新个人信息

1. 在个人中心页面找到需要修改的信息字段
2. 修改相应内容
3. 点击「保存」提交更改

### 步骤四：重置登录密码

1. 在个人中心页面找到「密码管理」区域
2. 点击「重置密码」按钮
3. 系统将自动生成一个符合安全要求的新密码
4. **务必立即复制并保存新密码**
5. 点击「确认」完成密码重置

## 常见问题

### Q1：忘记密码怎么办？

**A：** 如果您忘记了密码，请联系管理员为您重置密码。管理员可以通过用户管理功能为您生成新密码。

### Q2：密码重置后，旧密码还能用吗？

**A：** 不能。密码重置后，旧密码立即失效，只能使用新密码登录。

### Q3：为什么我看不到个人中心选项？

**A：** 个人中心对所有登录用户开放。如果您看不到该选项，请尝试刷新页面或联系技术支持。

### Q4：随机生成的密码太复杂，记不住怎么办？

**A：** 您可以再次重置密码，使用您自行设置的密码（需确保符合安全要求）。或者使用密码管理工具来存储密码。
