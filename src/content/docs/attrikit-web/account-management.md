## 广告账户管理

在管理后台中，您可以轻松管理所有广告平台的账户配置。

## 添加广告账户

### 支持的平台

目前支持以下广告平台：

| 平台 | 状态 | 说明 |
|------|------|------|
| 抖音 | ✅ | 包含巨量引擎 |
| 巨量引擎 | ✅ | 字节跳动广告平台 |
| 百度 | 🚧 | 即将支持 |
| 快手 | 🚧 | 即将支持 |
| 小红书 | 🚧 | 即将支持 |

### 添加步骤

1. 进入 **账户管理** 页面
2. 点击 **添加账户** 按钮
3. 选择广告平台
4. 填写账户信息：
   - 账户名称（自定义）
   - 广告账户 ID
   - API 凭证（Access Key / Secret Key）
5. 点击 **保存**

## 账户配置

### 基本设置

每个账户可配置以下参数：

- **账户名称**：用于识别的显示名称
- **时区设置**：确保数据时间准确性
- **货币设置**：数据展示的货币单位

### 数据同步

- **同步频率**：支持每小时、每日同步
- **历史数据**：首次接入可回溯 30 天历史数据
- **同步状态**：实时查看同步进度和结果

## 账户监控

### 同步状态

在账户列表中，每个账户会显示：

- ✅ **正常**：数据同步正常
- ⚠️ **警告**：同步延迟或异常
- ❌ **失败**：同步失败，需检查配置

### 数据验证

添加账户后，系统会自动验证：
1. API 凭证有效性
2. 账户访问权限
3. 数据可用性

## 常见问题

### API 凭证在哪里获取？

请前往各广告平台的开发者中心获取：
- **抖音/巨量引擎**：巨量引擎开放平台
- **百度**：百度营销开发者中心

### 一个账户可以添加多个广告账户？

可以，支持添加多个同平台的广告账户。

### 如何删除账户？

在账户列表中，点击操作列的 **删除** 按钮即可。删除后历史数据将保留 7 天。
