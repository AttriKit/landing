AttriKit Web 后台是整个 AttriKit 归因系统的核心管理平台，为用户提供监测链接创建、广告渠道管理、用户权限控制等功能。在完成私有化部署后，用户可以通过自行配置的自定义域名访问 AttriKit Web 后台。

AttriKit Web 后台提供了一站式的广告归因管理能力，帮助您高效管理监测链接、应用数据和用户权限。

**在这之前**:

> 请确保您已经完成了 AttriKit 系统的私有化部署，并获取了访问域名和初始管理员账号。如需帮助，请联系您的技术支持团队。

## 核心功能

- **监测链接管理** - 创建和管理曝光链接、点击链接
- **应用管理** - 添加和管理移动应用，获取应用 ID 用于 SDK 集成
- **用户管理** - 团队成员账号管理，支持多角色权限控制
- **个人中心** - 管理个人账号信息和安全设置

## 快速操作指南

### 步骤一：登录 AttriKit Web

使用初始管理员账号登录系统：
- 访问您的 AttriKit Web 域名
- 输入用户账号和密码
- 点击「登录」按钮

### 步骤二：添加应用

在开始创建监测链接之前，您需要先添加应用：

1. 点击右上角个人头像下拉菜单中的「应用管理」，进入应用管理页面
2. 点击「添加应用」按钮
3. 填写应用信息（应用名称、应用 ID、应用平台等）
4. 保存

### 步骤三：创建监测链接

添加应用后，即可创建监测链接：

1. 进入「监测链接管理」页面
2. 选择关联的应用，点击「新建监测链接」按钮
3. 填写链接名称、选择优化师
4. 确定生成

### 步骤四：添加团队成员（可选）

如需要多人协作，可以添加团队成员：

1. 点击右上角个人头像下拉菜单中的「用户管理」，进入用户管理页面
2. 点击「添加用户」按钮
3. 填写用户信息和选择角色
4. 保存后用户即可使用账号登录

## 权限系统

AttriKit 采用基于角色的访问控制（RBAC）系统，确保数据安全和权限隔离。

**角色说明**

| 角色 | 权限范围 | 说明 |
|------|----------|------|
| **Admin（管理员）** | 所有功能 | 拥有系统最高权限，可以管理用户、应用和所有监测链接 |
| **User（普通用户）** | 基础功能 | 可以管理拥有权限的应用下的所有监测链接，不可管理用户和应用 |

## 常见问题

### Q1：忘记管理员密码怎么办？

**A：** 如果您忘记了管理员密码，请联系技术支持团队进行密码重置。建议首次登录后立即修改密码。

### Q2：可以同时管理多个应用吗？

**A：** 可以。AttriKit 支持管理无限数量的应用，每个应用都有独立的应用ID和配置。

### Q3：监测链接有数量限制吗？

**A：** 没有。您可以根据需要创建任意数量的监测链接。

### Q4：如何修改个人密码？

**A：** 登录后，点击右上角的用户头像，选择「个人中心」，在密码修改区域输入新密码即可。
