## AttriKit 是什么？

AttriKit 是面向国内广告主的**第一方广告归因系统**，帮助你快速搭建属于自己的归因能力，摆脱对广告平台和第三方归因服务的依赖，获得数据的完全掌控权。

### 核心定位

- **不是 SaaS 服务**：我们提供可部署的系统代码，你自行部署在自己的服务器上

- **开箱即用**：无需专业开发团队，普通运维人员即可完成部署

## AttriKit 部署指南

### 部署前准备

**服务器要求：**
- CPU: 2 核心以上
- 内存: 4GB 以上
- 存储: 50GB 以上
- 系统: Linux (推荐 Ubuntu 20.04+)

**网络要求：**
- 固定公网 IP
- 开放 80、443 端口

### 快速部署（4 步完成）

**第 1 步：获取部署包**
从官方渠道获取 AttriKit Docker 部署包

**第 2 步：配置系统**
```bash
# 编辑配置文件
vim config.yaml

# 配置 API 密钥
# 配置数据库连接
# 配置管理员账号
```

**第 3 步：启动服务**
```bash
# Docker 一键启动
docker-compose up -d
```

**第 4 步：验证部署**
- 访问管理后台
- 连接广告平台账号
- 配置归因规则

**整个流程最快 1 小时即可完成。**

### SDK 集成（以 Android 为例）

```gradle
// 添加依赖
implementation 'com.attrikit:sdk:1.0.0'

// 初始化 SDK
AttriKit.init(context, "YOUR_API_KEY");

// 上报激活事件
AttriKit.trackEvent("activation");
```

### Q: 自建归因系统维护成本高吗？

### Q: 归因数据能导出吗？

### Q: AttriKit 合规吗？
